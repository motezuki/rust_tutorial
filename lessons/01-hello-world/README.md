# Lesson 01: Hello World

## Objective
Learn how to create and run your first Rust program.

## Concepts Covered
- The `main` function as the entry point
- Using `println!` macro for output
- Basic program structure

## Running the Lesson
```bash
cargo run
```

## Expected Output
```
Hello, world!
Welcome to Rust programming!
```

## Key Points
- Every Rust program starts with a `main()` function
- `println!` is a macro (note the `!`) that prints text to the console
- Rust uses semicolons to end statements
